## QS Pulldown Tweak

This is an extremely simple Xposed module that allows you to fully expand the quick settings UI with a single swipe down from either the left or right side of the status bar. I wrote this because it was the one feature of the [Dark Tricks Xposed module](https://github.com/darkeyes84/dark_tricks) that I missed, and that module hasn't been updated beyond Android 14. I ported the code for this from that module and modified it to make it work on Android 16+ (Android 15 might work too but is untested).
